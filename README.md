## Getting started

```
mvn install
mvn test
```

## Setup

Install the JDK

```
sudo apt-get install openjdk-11-jdk
```

Install Maven

```
sudo apt update
sudo apt install maven
```

Lombok

Add the Lombok extension to your IDE

Wildfly

Use RedHat addon to download, install and run WildFly.
